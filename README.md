## Ideas diseño slides

* [Que cumplan 4 reglas](https://speakerdeck.com/mseckington/the-art-of-slide-design): 

    * Un propósito por slide y no distraer

    * Resaltar información relevante

    * Usar texto e imágenes

    * Ser consistente
